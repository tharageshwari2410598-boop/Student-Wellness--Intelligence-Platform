# Student Wellness Intelligence Platform

An AI-inspired intelligent wellness monitoring system developed using Zoho Creator.  
This platform helps track student stress, sleep, and workload patterns while providing faculty with burnout risk insights.

---

##  Key Features

- Student stress & sleep tracking
- Burnout risk prediction indicator
- Faculty monitoring dashboard
- Animated and user-friendly interface
- Built-in Stress Relief Mini Game
- Data visualization and analytics
- Role-based access (Student / Faculty / Admin)

---

##  Live Application

🔗 https://creatorapp.zoho.in/tharageshwari2410598_ssn/student-wellness-intelligence-platform/

---

## 🛠 Technologies Used

- Zoho Creator
- Deluge Scripting
- Custom UI Components
- Role-based Portal Permissions

---

## 💡 Innovation Highlight

Unlike basic student record systems, this application focuses on predictive wellness intelligence and mental health awareness, making it impactful and future-oriented.
